Hypothesis
====
- H(x)=Wx+b
- H(x1,x2,x3)= w1x1 + w2x2 + w3x3 + b
    - 변수가 늘어난 만큼 가중치 늘어남. 일일히 써 주는 것이 불편해짐.
        - Matrix로 간단하게 표현

Matrix multiplication
===
선형대수학때 했던 matrix multiplication.

Hypothesis using matrix
---
### H(x)=XW
- matrix의 행, 열을 X, W로 표현
- X matrix를 입력받아 XW라고 표현함.
- X를 W앞에 쓴 이유 : 앞에 오는 matrix행, 뒤에오는 열을 연산하기때문에... X matrix가 앞에 오게.
